## Film

|   Service   |  Port  |         Description          |
|   :------   |  :---  |  :-------------------------  |
|   `Film`    |  8080  |        Gère les films        |

|  Route  |        Description           |
|   :--   |   :---------------------     |
|   `/`   |    Récupère tous les films   |
| `/{id}` |    Récupère un seul film     |
